# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-10

### Added
- **Dataflow Modeling System**: New comprehensive dataflow modeling capabilities for defining and managing ML workflows
- **RoPE Kernel**: Rotary Position Embedding (RoPE) hardware kernel implementation
- **Kernel Integrator**: System for integrating custom hardware kernels
- **Project-based Configuration**: New environment management system with project-scoped configurations
- **Modern CLI**: Complete CLI overhaul with improved user experience
- **Pydantic Configuration**: Type-safe configuration system using Pydantic models
- **Poetry Workflow**: Integrated Poetry for dependency management and packaging
- **Integration Tests**: Comprehensive integration test suite
- **Developer Guide**: Extensive developer documentation structure
- **Hardware Kernels Documentation**: Detailed documentation for custom kernel development
- **MLO Documentation**: Multi-Layer Offload system documentation

### Changed
- **Unified Registry**: Refactored to flat architecture with modern type system
- **Configuration System**: Complete refactor from legacy config to project-based environment management
- **Code Quality**: Applied Ruff formatting and linting across entire codebase
- **Documentation Structure**: Reorganized and polished all documentation
- **Import System**: Cleaned up dataflow imports to match linting standards
- **Test Organization**: Updated test filenames for better organization

### Fixed
- Multiple CI/CD pipeline improvements and fixes
- Documentation deployment workflow issues
- Component registration unused imports handling

### Technical Details
- Total changes: 248 files modified, 10,267 insertions, 8,707 deletions
- Code quality improvements via Ruff (formatting + automated lint fixes)
- Documentation polish and reorganization throughout project
