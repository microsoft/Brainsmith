### ğŸ”‘ Key Design Principles

1. **Type Safety Through Separation**

   - Inputs have configurable SDIM (streaming dimensions)

   - Uses QONNX's `BaseDataType` exclusively
   - Constraint-based validation with `DatatypeConstraintGroup`
   - No custom type system to maintain

   - Replaces ambiguous `iPar` with per-dimension streaming control
   - Only inputs have configurable SDIM
   - Relationships propagate SDIM between inputs
   - Outputs compute rates from kernel behavior
