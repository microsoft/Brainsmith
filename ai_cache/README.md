# AI Cache

This directory contains development artifacts and temporary files used during AI-assisted development.

## Structure
- `notes.md` - Simple notes and observations during development
- `issues/` - Ongoing issues and future tasks
- `designs/` - Design documents and initial implementation plans
- `plans/` - Final implementation plans with task checklists
- `reports/` - Analysis and post-implementation reports
- `tests/` - Temporary tests for validation and debugging
- `output/` - Test outputs (subfolder per specific test)

Note: This directory is for development artifacts only. Permanent tests and documentation belong in the main project structure.
